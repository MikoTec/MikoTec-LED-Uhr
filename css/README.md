# CSS-Dateien (ab v2.2.0.0)

Dieser Ordner enthält die Stylesheets für das LittleFS-Dateisystem des ESP8266.

## Geplante Dateien
- `style.css` — Haupt-CSS (bisher css.h / clockmenustyle.css)
- `menu.css` — Menü-Overlay Styles (bisher Teil von css.h)

## Konzept
CSS wird direkt aus dem Flash geliefert, kein Inline-CSS mehr im HTML.
